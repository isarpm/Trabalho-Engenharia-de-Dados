# Trabalho-Engenharia-de-Dados

Projeto desenvolvido para a disciplina de Engenharia de Dados, sob orientação do professor Andre Britto de Carvalho.  
O trabalho consiste em aplicações de CRUD em bancos relacionais e NoSQL, além da integração de dados em esquema estrela.

## Estrutura do Projeto

- **Parte 1 - CRUD Relacional (PostgreSQL/AWS)**
- **Parte 2 - CRUD NoSQL (MongoDB/AWS)**
- **Parte 3 - Integração de Dados (ETL para esquema estrela)**

## Organização do Repositório
```text
Trabalho-Engenharia-de-Dados/
│
├── docs/                Relatório final, diagramas e discussões de mapeamento
├── parte1-relacional/   Projeto Java (Maven) para o CRUD em PostgreSQL (AWS)
├── parte2-nosql/        Projeto Java (Maven) para o CRUD em MongoDB (AWS)
├── parte3-integracao/   Scripts e rotinas de integração (Esquema Estrela)
└── testes/              Testes automatizados (JUnit) para validação das operações
```
---

## Tecnologias Utilizadas

- **Java** (linguagem principal)
- **PostgreSQL** (relacional, hospedado na AWS)
- **MongoDB** (NoSQL, hospedado na AWS)
- **JUnit** (testes automatizados)
- **Maven/Gradle** (gerenciamento de dependências)
